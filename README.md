# sd-webui-custom
给[Stable Diffusion web UI](https://github.com/AUTOMATIC1111/stable-diffusion-webui)加点料

## 功能
隐藏了一些功能和设置入口   
预置了一些参数合集，方便快速起步   
![](./screenshot_1.jpg)

## 安装
把`custom_ui.js`丢到`javascript`目录，`user.css`丢到根目录，webui会自动加载
